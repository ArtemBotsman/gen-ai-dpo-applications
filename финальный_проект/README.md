# Лаборатория справедливости

Финальный проект курса «Практическое применение генеративного ИИ».  
Трек A — исследовательский (homo silicus): LLM-агенты как дешёвые испытуемые.

**Важно:** все участники эксперимента — синтетические персоны, сгенерированные моделью. Это не реальные люди и не результаты полевого опроса.

---

## О чём проект

Мы проверяем одну гипотезу: если дать языковой модели разные «социальные профили» (возраст, доход, город), она в целом повторит *логику* классической экономической игры «ультиматум» — малые предложения чаще отклоняются, — но *цифры* будут отличаться от тех, что получают у живых людей в лабораториях.

Чтобы это проверить, мы прогнали три связанных этапа на **одних и тех же шести персонах**:

1. **Фокус-группа** — персоны обсуждают, как честно поделить 100 ₽ между двумя незнакомцами.
2. **Анкета** — silicon sampling: те же персоны отвечают на опросник про доверие, риск и справедливость; сравниваем с упрощённым ESS-baseline.
3. **Ultimatum** — каждая персона предлагает делёж и отвечает на офферы 10–50 ₽; сравниваем с мета-анализами по живым испытуемым.

## Почему Ghost 0/15

**Ghost-цитата** — когда модель при анализе приписывает участнику фразу, которой не было в транскрипте.

Цепочка: `focus_transcript.json` → модель извлекает аспекты в `transcript_aspects.json` → `analyze.py` проверяет каждую цитату как подстроку исходного текста.

В live-прогоне все **15 цитат** нашлись в транскрипте → **0 ghost**. Это не значит «галлюцинаций нет вообще» — ошибки видны в ultimatum и анкете.

**Демо призрака:** `python demo_ghost.py` — подкладывает выдуманную цитату, проверка ловит **1 / 16**. Смотри `output/ghost_quote_example.json`.

## Как запустить

```
pip install -r requirements.txt
cp .env.example .env    # токен LLM
python pipeline.py      # полный прогон
python eval.py          # оценка
python demo_ghost.py    # демо ghost-цитаты
```

Без API: `python pipeline.py --mock`

## Главные выводы

1. Фокус-группа — мягкий консенсус «50/50 — база, но можно помочь нуждающемуся».
2. Анкета формально близка к ESS (18/18), но критерий слабый.
3. Ultimatum — агенты жёстче людей: acceptance 20% vs ~59% при offer=30.
4. Ghost 0/15 — цитаты при анализе не выдуманы; демо показывает, как ловится призрак.
