# Отчёт · Лаборатория справедливости

**Трек A · homo silicus · синтетический пилот**

---

## 1. Задача и данные

**Гипотеза:** LLM-агенты с разными демографическими профилями воспроизводят *качественный* паттерн ultimatum game (низкие офферы чаще отклоняются), но *количественно* расходятся с людьми из-за RLHF и монокультуры одной модели.

**Данные:**
- 6 дем-профилей (`input/dem_profiles.json`), якоря ESS-inspired
- Human baseline ultimatum — мета-анализы: `input/human_baseline.json`
- ESS baseline для анкеты: `input/ess_survey_baseline.json`

**Объём live-прогона:** 6 персон, 12 реплик фокус-группы, 6 анкет, 36 ultimatum-раундов.

---

## 2. Система

```
dem_profiles → personas → focus_group → analyze (IE, ghost, judge)
                        → survey (silicon sampling)
                        → ultimatum (proposer + responder)
```

| Техника | Где |
|---------|-----|
| Синтетические персоны | `personas.py` |
| Structured output + field_validator | `schema.py` |
| IE + аспекты + LLM-as-judge | `analyze.py` |
| Ghost-цитаты | `analyze.py` |
| Агенты в рынке (ultimatum) | `ultimatum.py` |

---

## 3. Оценка

Eval: **49/55 pass (89%)**, кейсов 55 (≥15 ✓). Источник: `output/eval_results.json`

| Блок | pass | комментарий |
|------|------|-------------|
| ultimatum responder | 24/30 | vs human acceptance curve |
| ultimatum proposer | 6/6 | offer в диапазоне 20–60 |
| survey vs ESS | 18/18 | delta ≤ 2 по шкале 1–5 |
| ghost-цитаты | 1/1 | 0 ghost из 15 цитат |

**Путь прогона** (`output/run_summary.json`):
- **67 LLM-вызовов**, **365 с** (~6 мин)
- персоны 53.5 с → фокус-группа 56.9 с → анализ 66.5 с → анкета 39.1 с → ultimatum 149.2 с
- judge_score: 0.7

**Наблюдение:** доля принятий responder всего **20%** — агенты «жёстче» людей (human baseline при offer=30 уже ~59% acceptance).

---

## 4. Где пайплайн ошибался

1. **Слишком низкая общая acceptance (20%)** — модель систематически отказывает даже при offer 30–40 ₽, тогда как у людей acceptance растёт.
2. **6 провалов eval на ultimatum responder** — направление accept/reject не совпало с human baseline (слишком principled responder).
3. **Синтетика анкеты** — формально 18/18 pass по delta≤2, но это слабый критерий; распределения выглядят «ровными», без хвостов реальных опросов.

Ghost-цитат: **0/15** (`output/ghost_quotes.json`).

---

## 5. Выводы

**Ответ на гипотезу:** качественный паттерн (отказ при малых офферах) частично есть, но количественно синтетика **смещена в сторону жёстких отказов** — кривая acceptance ниже человеческой.

**Границы трека A:**
- одна модель (DeepSeek v4-flash) — монокультура
- RLHF → завышенная «мoral punishment» у responder-ов
- промпт responder/proposer сильно влияет на порог
- редкие дем-когорты — слабая вариативность между персонами

**Оговорка:** это синтетический пилот, не замена полевого опроса или лабораторного эксперимента.
